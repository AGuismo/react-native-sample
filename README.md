# react-native-sample
Simple mobile app using React Native with Expo.


## Usage

```sh
  $ yarn && yarn start
```

## Note

In order to use Google Places API please edit `src/GMap.js` and add your own API Key:
```js
  const GOOGLE_PLACES_API_KEY = 'paste-your-key-here'
```
